# lintly

Linting tool for JavaScript projects.

## Installation

**npm**

```bash
npm install lintly
```

**yarn**

```bash
yarn add lintly
```

## Usage

> Note: In the below examples you can add multiple files/paths as separate arguments.

**npm**

```bash
npm run lintly /path/to/files/to/lint
```

**yarn**

```bash
yarn run lintly /path/to/files/to/lint
```
